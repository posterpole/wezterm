## Installation
* Create `colors/` directory in wezterm config path `$XDG_CONFIG_HOME/wezterm/`
* Copy posterpole.toml in wezterm config directory 
* Add color_scheme line in your wezterm config:

```lua
config.color_scheme = "posterpole"
```
